# Food Engines Submission Blurb

## Title

Food Engines

## Short Description

Food Engines is a bright cartoon farm-grid strategy game about food independence. Plant crops, build the systems behind a fed town, and discover that tomatoes are infrastructure.

## Theme Connection

This is an Independence Engine because food freedom is not just planting a crop. It is the connected system that keeps people fed: soil, water, seeds, labor, storage, kitchens, preservation, markets, weather response, and community trust.

## Suggested Submission Copy

Food Engines is a cartoon farm-grid strategy game about food independence. Players plant crops, harvest food, build water/soil/seed/kitchen/preservation/market engines, and answer town requests for school lunches, diner pies, firehouse chili, pantry drives, and county fair meals. It looks like a cheerful casual farm game, but underneath the tomatoes is a civic engine: can this little town feed itself when weather, spoilage, demand, and late grocery trucks start pushing back?
